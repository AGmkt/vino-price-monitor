💡 Come aggiungere un nuovo brand

1. Cerca il brand sull'e-commerce. Nota lo slug dall'URL: /produttore/nome-brand
2. Apri vino_scraper.py e aggiungi in BRAND_CONFIG:
"NOME BRAND": { "slugs": ["nome-brand"] },
3. Per sub-brand:
"SUB-BRAND": { "slugs": [], "sub_brand": True, "parent": "padre", "filter": r"(?i)nome" },
4. Committa, pusha, poi: Actions → Run workflow
